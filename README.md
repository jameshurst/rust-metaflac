A library for reading and writing FLAC metadata.

## Usage

Add the dependency to your `Cargo.toml`:

```toml
[dependencies]
metaflac = "0.2.8"
```

```rust
use metaflac::Tag;

fn main() {
  let tag = Tag::read_from_path("music.flac").unwrap();

  // Some things modifying the tag

  tag.save().unwrap();
}
```
